Writing Notes

Every note is a small HTML fragment inside the notes directory. The sidebar is generated from navigation.json.

Update an existing page

Open its HTML file and edit the content. For example, this page lives at notes/writing-notes.html.

Add a page

  1. Create an HTML file inside notes or a module subdirectory.
  2. Add the page to the appropriate module in navigation.json.
  3. Refresh the browser.

A page entry looks like this:

{
  "id": "java/strings",
  "title": "Strings",
  "file": "notes/java/strings.html"
}

Add a module

Add another module object to navigation.json:

{
  "title": "Databases",
  "pages": [
    {
      "id": "databases/overview",
      "title": "Overview",
      "file": "notes/databases/overview.html"
    }
  ]
}

Useful Bootstrap styles

Since Bootstrap is already loaded, notes can use components such as alerts, tables, badges, and cards directly.

Keep note files as HTML fragments. Do not include <html>, <head>, or <body> tags.
